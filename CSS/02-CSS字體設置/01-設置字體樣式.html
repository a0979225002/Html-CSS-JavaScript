「阿爾貝利亞競技大亂鬥」將於2020/11/04 14:00起變更可龍化的龍族，並進行平衡調整。

■變更可龍化的龍族
將無視武器種類，一律龍化為「邱比特」。
※今後可龍化的龍族可能會有所變更。
變更前：耶夢加得
變更後：邱比特
■平衡調整
分析目前統計的對戰資料後，確認到武器種類「斧」的性能勝率明顯高於其他武器。
為避免勝率差異過大，營運團隊已進行平衡調整。
【武器種類：斧】
  • 下調HP初始值，以及升級時的HP提升量。
  • 下調升級時的攻擊力提升量。
  • 下調武器種類專用技能威力。
■注意事項
1. 「阿爾貝利亞競技大亂鬥」之詳細，請參考活動任務頁面的「遊戲方式」內說明。
2. 如活動於戰鬥中結束，該戰鬥將繼續進行直到分出勝負。依據戰鬥的勝敗結果可獲得相對應的報酬。
